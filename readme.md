# Modelado 3D - Assets y recursos de clase (Blender)

Este repositorio reúne **assets, enlaces y recursos** que iremos mencionando en clase de **Modelado 3D** para tenerlos **centralizados, accesibles y reutilizables** desde GitHub.

La idea es simple: **entrar, encontrar lo que toca usar y descargarlo/consultarlo rápido**, sin perder tiempo buscando entre chats, PDFs o páginas sueltas.

---

## Sobre la asignatura

**Modelado 3D** es una asignatura optativa de **4º curso del Grado en Ingeniería Informática**, dentro de la línea de computación. Está orientada a aplicar conocimientos de computación a un campo muy visual, creativo y en expansión: el **diseño y modelado tridimensional**.

La asignatura combina:
- base **matemática** (superficies, interpolaciones, curvas)
- aplicación **práctica** en proyectos creativos
- herramientas de **modelado 3D** usadas en contextos reales

Aunque se recomienda haber cursado **Gráficos por Computador**, no es obligatorio. Con las bases matemáticas típicas de **Álgebra, Cálculo o Análisis Matemático** se puede seguir sin problema la parte teórica.

---

## Enfoque y herramientas

Durante el curso trabajaremos con **Blender** (software libre de creación 3D) como laboratorio de experimentación. A través de Blender exploraremos conceptos clave como:

- **Curvas paramétricas**
- **Superficies paramétricas**
- Generación y manipulación precisa de geometría
- Herramientas avanzadas para **geometrías complejas**
- Optimización de modelos para uso real

Además, se aborda el flujo práctico para:
- producir un modelo **optimizado**
- exportarlo a **STL**
- dejarlo listo para **impresión 3D** o revisión en **realidad virtual**
